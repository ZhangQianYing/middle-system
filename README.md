# middle-system

中台系统



- public :  基础头
- src
  - assets： 静态资源
  - components：组件
  - global： 通用数据，定义头部标题内容
  - router： 路由
  - store： vuex 公共数据
  - utils:  公共的js文件
    - axios.js : 请求数据
    - lib.js: 配置域名
    - valicdate.js:  公共的方法
  - views; 业务页面
  - main.js: 配置页
  - 。gitignore: 配置不上传项
  - package.json:  配置依赖
